# hello-rust

simple Rust app that generates and then solves a Sudoku puzzle...

### Dependency

[![Crates.io](https://img.shields.io/crates/v/sudoku?label=crates.io%20-%20sudoku)][sudoku-io]

### Usage:

`cargo run`
